# About

Simple financial system, made in Nodejs, without bank integration and data writing via fs (core module).

## Install

```bash
$ git clone https://github.com/RubenFontes/simple-financial-system 
```

```bash
$ cd simple-financial-system
```

```bash
$ npm install
```

## Initialize

```
$ npm start
```
